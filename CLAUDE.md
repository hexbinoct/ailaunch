# CLAUDE.md

Project-specific instructions for Claude Code sessions in this repo.

## At the start of every session

Read `IDEAS.md`. After reading, briefly mention (one or two sentences) which item from the list might be a good fit to work on next given the current state of the repo, recent commits, or whatever the user just brought up. Do not start implementing — just surface the suggestion so the user can decide.

If `IDEAS.md` is empty or every entry has been done, say so and move on.
